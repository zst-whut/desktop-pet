# Mochi - Desktop Pet

A cute canvas-rendered desktop cat pet. It walks around, blinks, talks, and reacts to your touch.

## Features

- **Auto walk** - Mochi wanders around the screen on its own
- **Click to pet** - Hearts fly out, happiness goes up
- **Double-click to feed** - Food appears, hunger refills
- **Drag to move** - Place Mochi anywhere you like
- **5 states** - Idle, Walking, Happy, Sleeping, Hungry
- **Speech bubbles** - Occasionally says cute things in Chinese
- **Status panel** - Hover to see happiness and hunger bars

## Usage

Open `index.html` in any browser, or host on GitHub Pages:

1. Fork this repo
2. Settings > Pages > Source: `main` branch
3. Visit `https://<username>.github.io/desktop-pet`

## Tech

Pure HTML/CSS/JS, Canvas 2D rendering, zero dependencies.
